# Form Identitas Mahasiswa
Aplikasi ini mengimplementasikan penggunaan XML Layout (Nested Layout), ScrollView, serta validasi input (View Validation) menggunakan bahasa pemrograman Kotlin.

**✨ Fitur Aplikasi**

🔹 Input Data Dinamis

Aplikasi menyediakan form input yang terdiri dari:
Nama (EditText)

Jenis Kelamin (RadioGroup & RadioButton)

Hobi (CheckBox)

🔹 Validasi Form

Mencegah input nama kosong dengan menampilkan error langsung pada EditText

Menampilkan Toast jika pengguna belum memilih jenis kelamin

🔹 Pemrosesan Data

Mengambil semua input dari user

Menampilkan kembali data dalam format rapi pada TextView hasil setelah tombol ditekan
